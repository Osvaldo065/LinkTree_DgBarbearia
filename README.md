# 💈 DG Barbearia

Bem-vindo à página oficial da **DG Barbearia** - A melhor barbearia da região!

## 📋 Sobre o Projeto

Este projeto é uma landing page moderna e responsiva para a barbearia DG, desenvolvida com HTML, CSS e Tailwind CSS. A página fornece informações sobre os serviços e oferece links diretos para contato via WhatsApp, Instagram e localização no Google Maps.

## 🎯 Funcionalidades

- ✅ Design responsivo (mobile-first)
- ✅ Interface intuitiva e moderna
- ✅ Links diretos para redes sociais e agendamento
- ✅ Efeitos visuais (hover e active para mobile)
- ✅ Background com imagem de fundo
- ✅ Paleta de cores em branco gelo

## 📁 Estrutura do Projeto

```
Project_DG/
├── index.html              # Arquivo principal da página
├── README.md               # Este arquivo
└── Assets/
    └── img/
        ├── home-pg.webp    # Imagem de fundo
        ├── Imagem_Perfil.jpg  # Foto de perfil da barbearia
        ├── icons8-whatsapp-50.png      # Ícone WhatsApp
        ├── icons8-instagram-50.png     # Ícone Instagram
        └── icons8-localização-50.png   # Ícone Localização
```

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização
- **Tailwind CSS** - Framework de utilidades CSS
- **WebP/JPG** - Otimização de imagens

## 📱 Recursos Responsivos
